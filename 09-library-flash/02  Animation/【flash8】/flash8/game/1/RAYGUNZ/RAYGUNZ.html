Notes:
The Raygunz game was developed for one purpose: to test how well a lot of graphics could be made to work in Flash. There are a number of separate tutorials that make this work... or you could just download this single file, in which all the following are put together:

(a) Square Root Function
(b) Detect Mouse Angle
(c) Game Gun
(d) Laser Collision Checking
(e) Terrain Scrolling

There are also two other "insignificant" little interface tutorials for the DOS Text Typer and the Value Slider.

This file is extensively commented and full of notes to explain the theory behind its working.

I will not provide any technical support on it. It is for educational purposes. Use it as you like but don't ask me for help! I had to stop development on this game because I got an actual job. So its relatively unfinished state you can blame on my new employer.
