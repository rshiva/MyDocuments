HOW TO CALCULATE THE SQUARE ROOT OF A NUMBER IN FLASH
One simple mathematical function oddly missing from the Flash Expression Editor abilities is the Square Root function. Since one wasn't there, and I HAD to have one, I went out and learned how to make one.

 

Here is a site that shows 3 different methods for calculating a square root.
http://www.cs.waikato.ac.nz/~rbg/sqrt/newton.html

I chose the Newtonian Method and made a function that copies it exactly. HOW the function works is not as important as finding a formula that you can plug into Flash. To be honest, I don't fully understand the function. I'm no mathemetician. But it works.

Since this method ESTIMATES the square root of a number, you may get funny results. Try entering a "0".

Still, for purposes in  Flash, it gets close enough!

To reuse this function in your own calculations, simply copy the frame called "sqrt" into your document. You need to set the variable "iterations" in your file as well. Set it to 10 or 15. This is how many times Flash will cycle through the formula (each time it loops through the sqrt formula, the result gets closer to the actual square root of the number.)

The function ASSUMES you have a variable called "input". The function performs its magic on that variable, then finishes off by creating a variable called "output".

That's it.
